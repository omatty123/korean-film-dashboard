This page posts one attendance bundle per class to a Google Apps Script web app. The script then replaces that class meeting in your sheet instead of duplicating rows every time.

View payload shape 
{
  "course": "EAST 418 / FILM 419",
  "sessionId": "2026-03-30",
  "sessionDate": "2026-03-30",
  "sessionLabel": "Monday, March 30, 2026",
  "topic": "The Joseon Court",
  "sheetName": "Attendance",
  "notes": "optional note",
  "counts": { "present": 5, "late": 1, "absent": 1, "excused": 0, "unmarked": 0 },
  "students": [
    { "id": "jack", "name": "Jack", "status": "present" }
  ]
}

This makes the tool less public-facing, not private. The page is unlisted and marked noindex, but the repo and site are still public until you move it behind private hosting.
